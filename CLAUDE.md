# claude-tutorial プロジェクトルール

## 概要
Claude Code の学習用プロジェクト

## ファイル構成
- index.html → 花カウンター（メインページ）
- profile.html → 自己紹介ページ
- tutorial.html → チュートリアルまとめWebページ版
- hello.py → Python学習用スクリプト
- チュートリアルまとめ.md → 学習の記録とロードマップ

## 技術
- HTML / CSS / JavaScript（Webページ）
- Python（学習用スクリプト）

## ルール
- HTMLファイルはダークモードで統一する
- コミットメッセージは日本語で書く
- 新しい学習項目を完了したら tutorial.html とチュートリアルまとめ.md を更新する

## 禁止事項
- 既存ファイルを確認なしに削除しない
- 英語でコメントを書かない（日本語で書く）
- 外部ライブラリやフレームワークを勝手に追加しない
